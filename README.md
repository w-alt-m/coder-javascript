# Proyecto: Lista de Tareas (To-Do List)

Esta es una aplicación web de Lista de Tareas, creada para el curso de Javascript de Coderhouse. Permite agregar, editar y eliminar tareas, y persiste la información localmente usando LocalStorage. Adicionalmente, carga tareas iniciales a modo de ejemplo desde un archivo JSON.

## Tecnologías Utilizadas

- **HTML5**
- **CSS3** (Estilos modernos, Animate.css)
- **Javascript** (Lógica de la aplicación)
- **Bootstrap 5** & **Bootstrap Icons**
- **SweetAlert2** (Alertas personalizadas)

## Nota sobre el Desarrollo

La lógica de la aplicación (Javascript) fue desarrollada por mí.
Se utilizó asistencia de Inteligencia Artificial exclusivamente para el diseño de la Interfaz de Usuario (UI) y los estilos (CSS), con autorización del curso.
